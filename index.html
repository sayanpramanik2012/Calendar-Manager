Calendar Event Manager

Welcome to the Calendar Event Manager website. This application allows you to add events to your Google Calendar with ease.

To get started, make sure you have the necessary credentials file (credentials.json) and Python packages installed. You can refer to the project's GitHub repository for more information.

Once you have everything set up, run the Python script to launch the application and start managing your calendar events.

For detailed instructions and usage, please check out the README file in the project repository.

If you have any questions or need support, feel free to contact us at your-email@example.com.

Thank you for using the Calendar Event Manager!
